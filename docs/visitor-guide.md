# 1. はじめに ─ アプリのインストール

## アプリって何？

南高祭公式サイトは**スマートフォンにインストールして使うアプリ**として動作します。  
インストールするとホーム画面にアイコンが追加され、普通のアプリと同じように使えます。

> ✅ **スタンプラリーと通知機能を使うにはインストールが必要です**  
> マップや展示情報だけなら、ブラウザからそのまま使えます。

## インストール方法

### iPhone の場合（Safari を使用）

```
1. Safari でこのサイトを開く
   ※ Chrome など他のブラウザでは追加できません

2. 画面下部の「共有」ボタンをタップ
   （四角から上矢印が出ているアイコン）

3. 上にスクロールして「ホーム画面に追加」を選ぶ

4. 右上の「追加」をタップ

5. ホーム画面に「南高祭」のアイコンが表示されれば完了！
```

*[スクリーンショット: iOSの「ホーム画面に追加」手順]*

### Android の場合（Chrome を使用）

```
1. Chrome でこのサイトを開く

2. 画面上部または下部に「ホーム画面に追加」の案内が表示される
   ※ 表示されない場合は右上メニュー（⋮）→「ホーム画面に追加」

3. 「追加」をタップ

4. ホーム画面にアイコンが追加されれば完了！
```

# 2. 画面の見方

## タブバー（画面下部）

アプリを開くと画面下部に5つのタブが並んでいます。

```
┌────────────────────────────────────────┐
│                                        │
│            メインコンテンツ              │
│                                        │
├─────┬──────┬──────┬──────┬──────────┤
│🗺   │ 🔔   │ 📣   │ 🎯   │  📅     │
│マップ│ 通知  │お知らせ│スタンプ│  予定   │
└─────┴──────┴──────┴──────┴──────────┘
```

| タブ | できること |
|---|---|
| 🗺 マップ | 校内マップで展示を探す |
| 🔔 通知 | 展示のフォロー・通知設定 |
| 📣 お知らせ | 各クラスの最新情報を見る |
| 🎯 スタンプ | QR コードを読み取りスタンプを集める |
| 📅 予定 | 自分の当日スケジュールを管理する |

## ヘッダー（画面上部）

- **左**: 南高祭ロゴ（タップでトップページへ）
- **右**: 現在時刻（時：分：秒）
- **下段**: 各クラスからのお知らせがスクロール表示される

# 3. 🗺 マップ ─ 校舎を探索しよう

*[スクリーンショット: マップ全体表示（2Fの例）]*

## 基本操作

| 操作 | 動作 |
|---|---|
| 指2本でピンチ | 拡大・縮小 |
| 1本指でドラッグ | マップを移動 |
| 教室をタップ | その展示の情報を表示 |

## フロアを切り替える

画面左側のボタンで 1F〜6F を切り替えられます。

```
 [6F]
 [5F]
 [4F]
 [3F]
 [2F] ← 現在のフロア（強調表示）
 [1F]
```

## 展示を探す

画面上部の検索バーにクラス名や展示名を入力すると、マップ上でハイライトしてフロアも自動で切り替わります。

```
 ┌─────────────────────────┐
 │ 🔍 お化け屋敷           │
 └─────────────────────────┘
         ↓ 入力すると
   該当する教室がオレンジ色になる
   フロアも自動で切り替わる
```

## 展示情報を見る

教室をタップすると画面下から情報が表示されます。

*[スクリーンショット: ボトムシート（展示名・待ち時間・詳細ボタン）]*

```
┌──────────────────────────────────┐
│ 📍 201教室 · 2F                  │
├──────────────────────────────────┤
│ 🎨 高2-1  お化け屋敷             │
│ 種別: 展示                        │
│                                  │
│ 混雑状況 ━━━━━━━━━━░░  約15分     │
│                                  │
│ ❤ いいねする                     │
│                                  │
│ [  展示の詳細を見る →  ] [🔔]   │
└──────────────────────────────────┘
```

- **混雑状況バー**: 緑（待ちなし）→ 黄（〜10分）→ オレンジ（〜25分）→ 赤（それ以上）
- **「展示の詳細を見る」**: 写真・説明・コメントなど詳細ページへ
- **🔔ボタン**: この展示からのお知らせ通知を ON/OFF

各クラスや実行委員からのリアルタイム情報を確認できます。

## 一覧の見方

```

# 5. 🔔 通知 ─ 気になる展示をフォロー

*[スクリーンショット: 通知設定ページ]*

気になる展示をフォローして、新しいお知らせが届いたときにプッシュ通知を受け取れます。

## 通知を ON にする手順

```
1. 「通知」タブをタップ

2. 「南高祭 公式アナウンス」を ON にする
   （緊急連絡・全体案内を受け取れる）

3. 「許可する」ボタンをタップ
   ブラウザの確認ダイアログが出たら「許可」を選択

4. 気になるクラスの通知を個別に ON/OFF できる
```

## 展示ごとの通知設定

展示名のリストから通知したい展示を選んで「🔔 ON」ボタンをタップ。

```
┌──────────────────────────────────────┐
│ 🎭 展示・催し                         │
├──────────────────────────────────────┤
│  [写真] 高2-1 お化け屋敷    [🔔 ON ] │
│  [写真] 高2-2 縁日          [🔕 OFF] │
│  [写真] 高2-3 謎解き        [🔕 OFF] │
├──────────────────────────────────────┤
│ 🍱 フード                            │
├──────────────────────────────────────┤
│  [写真] 高3-1 焼きそば      [🔔 ON ] │
└──────────────────────────────────────┘
```

> **iOS をお使いの場合**: プッシュ通知を受け取るには PWA インストール（ホーム画面に追加）が必要です。

# 6. 🎯 スタンプラリー ─ 展示を巡ろう

> **ホーム画面への追加（PWA インストール）が必要です**

*[スクリーンショット: スタンプラリーのグリッド画面（一部押されている状態）]*

展示を訪れて QR コードを読み取ると、その展示のスタンプがもらえます。  
スタンプを集めると**いいね**と**人気投票**に参加できます。

## 参加の流れ

```
①  ホーム画面のアイコンからアプリを起動
        ↓
②  「スタンプ」タブをタップ
        ↓
③  「QR をスキャン」ボタンをタップ
        ↓
④  カメラが起動 → 展示のQRコードを読み取る
        ↓
⑤  スタンプが押される！グリッドに写真が表示される
```

## スタンプの画面

スタンプが押されると、展示の写真がグリッドに表示されます。

```
┌──────┬──────┬──────┬──────┬──────┐
│[写真]│[写真]│      │      │      │
│高2-1 │高2-3 │  未  │  未  │  未  │
├──────┼──────┼──────┼──────┼──────┤
│      │      │      │      │      │
│  未  │  未  │  未  │  未  │  未  │
└──────┴──────┴──────┴──────┴──────┘
      ↑ 押された展示の写真が表示
```

- ページが複数ある場合は横スワイプで切り替え
- コンプリートすると「🎉 完了！」と表示

## QR コードについて

- QR コードは**1分ごとに自動更新**される特殊なコードです
- 他の人の QR 画像の写真を使っても、最大2分後に無効になります
- 必ず会場で**直接**スキャンしてください

# 7. 🗳 人気投票 ─ お気に入りに投票しよう

**スタンプを1つ以上集めると投票できます。**

*[スクリーンショット: 投票ページ（展示リストと現在のランキング）]*

## 投票方法

```
1. 「スタンプ」タブ → スタンプを1つ以上集める

2. 下のタブバー... ではなく、
   マップ → 展示詳細 → 投票ページへのリンク
   または、直接 /vote にアクセス

3. スタンプを集めた展示の一覧が表示される

4. 投票したい展示を選んで「投票する」ボタン
```

## ランキング

現在のランキングが表示されます（管理者が公開設定した場合）。

```
🏆 現在のランキング

🥇  高2-1 お化け屋敷
🥈  軽音楽部
🥉  高2-3 謎解き
  4  高3-2 焼きそば
```

## 投票の変更

投票後も投票先を変更できます。一度入れた票を別の展示に移せます。  
（同時に複数の展示に投票することはできません）

# 8. 📅 予定 ─ 自分だけのタイムラインを作ろう

*[スクリーンショット: 予定タイムライン（8:30〜16:30、イベントが並んでいる状態)]*

当日訪れたい場所や見たいステージを**8:30〜16:30のタイムライン**で管理できます。

## 予定の追加方法

### 方法①：展示・イベントのページから追加（かんたん）

展示詳細ページ・軽音ページ・催し物ページに「📅 予定に追加」ボタンがあります。

```
1. 行きたい展示のページを開く

2. 「📅 予定に追加」ボタンをタップ

3. 通知タイミングを選ぶ（なし / 5分前 / 10分前 / 15分前 / 30分前 / 1時間前）

4. 「追加する」ボタン → 完了！
```

### 方法②：予定ページから自分で入力

```
1. 「📅 予定」タブをタップ

2. 右上の「＋」ボタンをタップ

3. 以下を入力：
   ・タイトル（例：「高2-1に行く」）
   ・日付（土曜日 / 日曜日）
   ・開始時刻・終了時刻
   ・場所（任意）
   ・通知タイミング

4. 「追加する」ボタン
```

## タイムラインの見方

```
8:30 ──────────────────
9:00 │ 🟠 高3-1 焼きそば │
     │  （クスノキ広場）  │
9:30 ──────────────────
     │ 🟢 カスタム予定   │
10:00──────────────────
                          ← 空き時間（何も表示しない）
10:30──────────────────
     │ 🟠 軽音・バンドA  │  │ 🟠 催し物・ダンス │
     │  （体育館）       │  │  （メインアリーナ）│
11:00──────────────────
```

- **🟠 オレンジ**: 展示・イベントの訪問予定
- **🟢 グリーン**: 自分で入力した予定
- **🟣 パープル（ログイン時）**: 自分のシフト当番
- 同じ時間帯に予定が重なると**横に並べて**表示

## 予定の削除

タイムライン上の予定をタップすると削除の確認ダイアログが表示されます。

## 土曜・日曜の切り替え

画面右上の「土」「日」ボタンで日付を切り替えられます。

## 通知について

設定した時間前にブラウザ通知でお知らせします。  
アプリを開いているとき（またはPWAとして起動中）に通知が届きます。

# 10. 🎸 軽音ライブ・催し物 ─ ステージスケジュール

*[スクリーンショット: 軽音ページ（バンド一覧・ライブ中バッジ）]*

```
── タイムライン ──────────────────────

       ← LIVE 演奏中！（オレンジのバッジ）

# 12. 困ったときは

## よくある質問

**Q. スタンプラリーが使えない**

A. ホーム画面への追加（PWA インストール）が必要です。  
iPhone なら Safari、Android なら Chrome からアクセスして「ホーム画面に追加」を行ってください。  
詳しくは [1. はじめに ─ アプリのインストール](#1-はじめに--アプリのインストール) を参照。

---

**Q. 通知が届かない（iPhone）**

A. iOS では PWA としてインストールした場合のみ通知が届きます。  
Safari の「ホーム画面に追加」からインストールして、ホーム画面のアイコンから起動してください。

---

**Q. QR コードを読んだのにスタンプが押されない**

A. 以下を確認してください：
- スタンプラリーのページが「スタンプ参加中」の展示の QR ですか？（すべての教室にQRがあるわけではありません）
- QR コードは1分ごとに変わります。撮影した写真では読み取れません。直接スキャンしてください。

---

**Q. 予定の通知が届かない**

A. 通知はアプリが開いているとき（またはバックグラウンドで起動中）に届きます。  
アプリをホーム画面に追加してお使いください。

---

**Q. マップが表示されない**

A. 文化祭の準備中などの期間はマップが一時的に非公開になっている場合があります。  
開催当日になると公開されます。

---

**Q. アプリのデータを削除したい**

A. スタンプや予定データはこの端末にのみ紐付いています。  
ブラウザの「サイトのデータを消去」または「キャッシュを削除」から削除できます。
