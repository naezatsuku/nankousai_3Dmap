## 1. サイトについて

### どんなサイトか

南高祭の来場者向け公式アプリサイトです。  
スマホで開いてホーム画面に追加すると、アプリのように使えます。

**URL：** https://nankousai-3d-map.vercel.app

### 主な特長

| 特長 | 説明 |
|------|------|
| 3D マップ | 校舎の各フロアを立体的に確認できる |
| リアルタイム更新 | 待ち時間やお知らせが自動で最新になる |
| プッシュ通知 | 催しの開始前などにスマホへ通知が届く |
| オフライン対応 | 一度開けば電波が弱くてもある程度使える |
| アプリとして使える | ホーム画面に追加するとアプリ感覚で使用可能 |

## 2. 来場者が使える機能

### トップページ（ `/` ）

サイトを開いたときの最初の画面です。  
マスコットキャラクター「nanpen（なんぺん）」のアニメーションが表示され、  
各ページへのメニューが並んでいます。

---

### 3D マップ（ `/map` ）

校舎の 3D マップでフロアを確認できます。

- **フロア切替**：画面左下のボタンで 1F〜6F を切り替えられます
- **ズーム・移動**：ピンチ操作やドラッグで拡大・移動できます
- **部屋タップ**：部屋をタップすると、その部屋の展示情報が下からスライドして表示されます
- **待ち時間カラー**：部屋の色で混雑状況が一目で分かります
  - 🟢 緑：空いている（0分）
  - 🟡 黄：少し混んでいる（〜10分）
  - 🟠 橙：混んでいる（〜25分）
  - 🔴 赤：かなり混んでいる（25分〜）
  - ⬜ グレー：展示なし
- **検索**：画面上の検索バーでクラス名や展示名を入力すると該当の部屋がハイライトされます

---

### お知らせ（ `/news` ）

運営からのお知らせ一覧です。  
未読のお知らせがある場合、タブバーにバッジが表示されます。  
ページを開くと自動的に既読になります。

---

### 通知設定（ `/notifications` ）

プッシュ通知の ON/OFF を設定するページです。  
（詳細は「[3. プッシュ通知](#3-プッシュ通知)」を参照）

---

### バンド（ `/band` ）

軽音楽部の出演スケジュールと各バンドの情報が確認できます。  
土曜・日曜で切り替えて表示できます。

---

### フード（ `/food` ）

フード販売の情報が確認できます。

---

### 催し物（ `/special` ）

ダンス・演劇・合唱など特別なパフォーマンスのスケジュールです。  
現在進行中の催しは「NOW ON STAGE」として目立つ表示になります。

---

### ヘッダー（常に表示）

画面上部のヘッダーには以下が表示されています。

- **南高祭ロゴ**：タップするとトップページに戻ります
- **現在時刻**：リアルタイムで更新されます
- **ティッカー（流れるテキスト）**：緊急連絡・アナウンス・開催中の催し情報が自動で流れます

## 3. プッシュ通知

スマホの通知機能を使って、大切な情報をお届けします。

### 通知の種類

| 種類 | タイミング | 内容 |
|------|-----------|------|
| 全体アナウンス | 随時 | 運営からの緊急連絡・全体お知らせ |
| 催し開始通知 | 開始10分前 | 「〇〇まもなく開始」 |
| 催し開始通知 | 開始時 | 「〇〇が始まりました」 |

### 設定方法

1. タブバーの「🔔 通知」をタップ
2. 「全体のお知らせ」の「許可する」をタップ → スマホの通知許可ダイアログで「許可」
3. 各団体のボタンで個別に ON/OFF できます

> **注意**：iPhone の場合、ホーム画面に追加した「PWA モード」でのみ通知が使えます。  
> Safari でそのまま開いているだけでは通知は届きません。

## 4. 管理者ができること

管理者はブラウザから `/admin` にアクセスして操作します。  
ログインには管理者アカウントが必要です。

### 管理メニュー一覧

| メニュー | できること |
|---------|-----------|
| ダッシュボード | 展示の一覧・待ち時間の概況 |
| 展示編集 | 各展示の待ち時間・情報を更新する |
| お知らせ管理 | お知らせの投稿・削除 |
| アナウンス管理 | ヘッダーのティッカーに流れる文章を管理、プッシュ通知送信 |
| 通知テスト | 催し開始通知をテスト送信する |
| 権限管理 | 管理者・編集者のアカウントを管理する |
| 団体管理 | 展示・催しの団体情報を管理する |

### 待ち時間の更新方法

1. 管理画面 → 「展示編集」
2. 該当の展示をタップ
3. 待ち時間（分）を入力して保存

更新すると **即座に来場者のマップに反映**されます。

### アナウンスの送信方法

1. 管理画面 → 「アナウンス管理」
2. 「＋ 新規作成」をタップ
3. テキストを入力（緊急の場合は「緊急」トグルを ON）
4. 「送信」ボタンを押すと、ヘッダーのティッカーへの表示 **＋ 通知登録者全員へプッシュ通知** が同時に届きます

### 権限の種類

| 権限 | できること |
|------|-----------|
| **admin（管理者）** | すべての操作 |
| **editor（編集者）** | 割り当てられた展示の待ち時間・情報の編集のみ |

## 5. データはどこに保存されているか

このサイトのデータは **Supabase**（スパベース）というクラウドサービスに保存されています。  
Google Drive のようなものをイメージしてください。

### 保存されているデータの種類

| データ | 説明 |
|--------|------|
| exhibits（展示） | 展示名・クラス名・待ち時間・フロア情報など |
| announcements（アナウンス） | ヘッダーティッカーに流れる文章 |
| notices（お知らせ） | お知らせ一覧の内容 |
| band_schedules（バンドスケジュール） | 軽音楽部の出演時間・ステージ情報 |
| special_schedules（催しスケジュール） | ダンス・演劇などのスケジュール |
| push_subscriptions（通知登録） | 通知を受け取るデバイスの一覧 |
| profiles（ユーザー） | 管理者・編集者のアカウント情報 |

### リアルタイム更新の仕組み

来場者がサイトを開いている間、データベースの変更を **自動で受信**します。  
管理者が待ち時間を更新すると、来場者の画面に **数秒以内**に反映されます。

## 6. スマホへのインストール方法（PWA）

このサイトはアプリとしてホーム画面に追加できます（無料・アプリストア不要）。

### iPhone（Safari）の場合

1. Safari でサイトを開く
2. 画面下の「共有ボタン」（四角に上矢印のアイコン）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

### Android（Chrome）の場合

1. Chrome でサイトを開く
2. 右上のメニュー（縦三点）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」をタップ

> インストール後はアプリのように全画面で開き、ホーム・ネット接続なしでも一定の機能が使えます。

## 7. サイトを更新・運用するには

### 費用について

| サービス | 用途 | 費用 |
|---------|------|------|
| Vercel | サイトの公開 | 無料プランで運用中 |
| Supabase | データベース・認証 | 無料プランで運用中 |
| Firebase | プッシュ通知 | 無料プランで運用中 |

現状すべて**無料プラン**で運用しています。  
大規模な来場者には対応できていますが、将来的に制限に達する場合は有料プランへの移行が必要になることがあります。

### コードの変更・デプロイ方法

1. コードを修正する
2. GitHub にプッシュする
3. Vercel が自動的に新バージョンを公開する（約 1〜2 分）

## 8. 困ったときは

### よくある質問

**Q. マップが表示されない**  
A. ページを下に引っ張って（プルダウンリフレッシュ）更新してみてください。  
それでも表示されない場合は、ページを閉じて再度開いてください。

**Q. 通知が届かない**  
A. iPhone の場合、Safari で直接開いているだけでは通知は届きません。  
ホーム画面に追加した状態（PWAモード）で設定してください。

**Q. 管理画面にログインできない**  
A. パスワードを忘れた場合は、ログインページの「パスワードを忘れた方」から再設定できます。

**Q. 待ち時間が更新されない**  
A. 管理者アカウントでログインし、「展示編集」から更新してください。  
更新後は来場者の画面に即時反映されます。
